# Trailer files

Export the trailer at 1920x1080, 30 fps, with a 16:9 aspect ratio.

Place both versions here:

- `marionette-trailer.webm` using VP9 video and Opus audio for the smallest file
- `marionette-trailer.mp4` using H.264 video and AAC audio as the compatibility fallback

Keep the video around 30-60 seconds and target 4-8 MB. The page already loads only metadata until the visitor presses play.
